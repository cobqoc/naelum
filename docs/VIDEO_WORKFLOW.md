**상태**: v0 (draft) — Day 4 LAUNCH 플랜의 일부
**목적**: 1인 개발자가 주 3회 유튜브 쇼츠를 지속 가능하게 찍는 파이프라인
**작성일**: 2026-04-16

## 🎯 영상 종류

### A. 인트로 영상 (1회만)
- 길이: 90~120초
- 내용: "낼름이 뭐하는 앱인가" + "왜 만들었는가"
- 스크립트: `docs/launch/intro_video_script.md`

### B. 레시피 쇼츠 (주 3회 목표)
- 길이: 45~60초 (유튜브 쇼츠 최적)
- 포맷: 세로 1080×1920
- 내용: 1 레시피 = 1 영상 (재료 → 핵심 단계 3컷 → 완성샷 → CTA)

### C. 다국어 변형 (선택)
- B를 영어 음성으로 재생성 → 동일 영상 #shorts #korean-food 태그
- 타겟: 재한 외국인, 해외교포

---

## 🛠 파이프라인 (B 레시피 쇼츠 기준)

```
[1] 레시피 선택        →  Supabase 쿼리 (조회수 + 난이도 필터)
       ↓
[2] 스크립트 생성       →  Gemini 2.5 (재료/단계 입력 → 60초 대본 출력)
       ↓
[3] 음성 합성 (TTS)     →  ElevenLabs / Google Cloud TTS
       ↓
[4] 비주얼 구성         →  레시피 썸네일 + 단계 사진 (현재 DB) + Pexels 보조컷
       ↓
[5] 영상 합성           →  CapCut / Remotion / FFmpeg
       ↓
[6] 자막 + 자동 BGM     →  CapCut 자동 캡션
       ↓
[7] 업로드              →  YouTube Studio (수동) → Studio API (자동화는 v2)
```

---

## 🔧 도구 선택 — 무료/저비용 우선

| 단계 | 도구 | 비용 | 비고 |
|------|------|-----|-----|
| 스크립트 | Gemini 2.5 Flash | 무료 (월 1500 req) | API 키 발급 |
| TTS (한국어) | Google Cloud TTS Neural2 | $4/1M chars | 월 $1 미만 예상 |
| TTS (영어) | ElevenLabs (free tier) | 무료 (월 10k chars) | 자연스러움 우선 |
| 사진 보조 | Pexels API | 무료 | 음식 스톡 풍부 |
| 영상 합성 | CapCut Desktop | 무료 | 모바일 앱도 가능 |
| 자막 | CapCut 자동 캡션 | 무료 | 한/영 정확도 OK |
| 음악 | YouTube Audio Library | 무료 | 저작권 안전 |

**핵심 원칙**: v1은 자동화에 욕심내지 말 것. 수동으로 5개 만들어보고 패턴 잡힌 다음에 자동화한다.

## ✍️ Gemini 스크립트 생성 프롬프트 (레시피 쇼츠)

```
당신은 60초 유튜브 쇼츠 대본 작가입니다.
아래 한국 요리 레시피를 60초 분량(약 150자)의 자연스러운 한국어 내레이션으로 변환하세요.

레시피 제목: {{title}}
조리 시간: {{cook_time}}분
재료: {{ingredients}}
단계: {{steps}}

요구사항:
- 후크(0~5초): "이 레시피, 단 X분이면 끝납니다"처럼 이득 + 시간을 강조
- 본문(5~50초): 핵심 단계 3개만 압축 (모든 단계 X)
- CTA(50~60초): "재료로 추천 받고 싶다면 낼름 앱에서" (반드시 포함)
- 어려운 한자어 금지, 친구한테 말하듯이
- 숫자는 한글로 ("2분"보다 "이 분", "3큰술"보다 "세 큰술")
- 마침표 단위로 줄바꿈 (TTS 호흡)
```

영어판은 동일 프롬프트에서 한국어→English로 바꾸고, "American friendly tone, US measurements when natural" 추가.

## 📐 영상 시각 템플릿

```
세로 1080×1920
┌──────────────────┐
│  [상단] 큰 자막   │  ← 핵심 키워드 (예: "5분 김치찌개")
│                  │
│   [중앙] 영상     │  ← 레시피 사진 / 단계 사진
│   1280×720       │
│                  │
│  [하단] 작은 텍스트│  ← 내레이션 자막
│  + 낼름 로고 워터 │
│  + 우하단 QR     │  ← naelum.app
└──────────────────┘
```

- 우하단 QR 코드 (naelum.app) 항상 표시 → 시청자가 모바일로 바로 스캔
- 워터마크: "낼름" 로고 좌상단 작게

## 🚦 주 3회 운영 체크리스트

매주 일요일에 한 번에 작업:
- [ ] 레시피 3개 선정 (Supabase에서 popular 정렬)
- [ ] Gemini로 3개 스크립트 일괄 생성
- [ ] TTS 음성 3개 생성
- [ ] CapCut에서 3개 영상 합성 (각 ~20분)
- [ ] 월/수/금 예약 업로드 (YouTube Studio)
- [ ] 인스타/Threads 같은 영상 동시 업로드

총 소요시간 목표: **2시간 / 주**
