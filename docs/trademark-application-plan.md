## 2. 출원 전략

### 2.1 출원 상표

```
한글: 낼름
영문: Naelum
```

### 2.2 출원 상품류 (니스 국제분류)

#### [필수] 제9류 — 소프트웨어/앱
> 다운로드 가능한 모바일 앱, 요리 레시피 관련 소프트웨어, 
> 스마트폰용 애플리케이션 소프트웨어

**지정상품 예시:**
- 다운로드 가능한 모바일 애플리케이션 소프트웨어
- 요리 레시피 검색용 컴퓨터 소프트웨어
- 음식 재료 관리용 컴퓨터 소프트웨어
- 레시피 공유용 소프트웨어

#### [필수] 제42류 — 소프트웨어 서비스/플랫폼
> SaaS, 웹 기반 플랫폼, 소프트웨어 개발

**지정상품 예시:**
- SaaS(서비스형 소프트웨어) 제공업
- 웹사이트를 통한 소프트웨어 제공업
- 클라우드 컴퓨팅을 이용한 소프트웨어 제공업
- 요리 레시피 데이터베이스 제공업

#### [권장] 제35류 — 온라인 플랫폼/마케팅
> 온라인 상거래 중개, 데이터베이스 관리

**지정상품 예시:**
- 인터넷을 이용한 광고업
- 온라인 정보 제공업 (레시피)
- 컴퓨터 데이터베이스를 통한 정보 편집업

#### [선택] 제43류 — 음식 관련 서비스
> 이미 "낼름" 등록 존재 (박윤채) → 거절 가능성 높음
> 불필요하면 출원하지 않음

### 2.3 출원 방식

| 방식 | 장점 | 단점 | 비용 |
|------|------|------|------|
| **특허로 셀프출원** | 비용 최소화 | 지정상품 작성 실수 리스크 | 관납료만 |
| 변리사 대행 | 전문가 검토, 거절 대응 | 비용 추가 | +20~40만원 |
| 온라인 대행 (마크인포 등) | 간편, 중간 비용 | 대면 상담 부족 | +10~20만원 |

**추천: 특허로 셀프출원 (2~3류 출원 시)**
- 비용 절감 최대
- 선행상표 조사 이미 완료
- 거절 시 의견서 제출은 변리사 별도 의뢰 가능

## 4. 출원 절차 및 타임라인

### 4.1 출원 전 준비 (1~2일)

- [ ] **특허고객번호** 발급 (특허로에서 온라인 신청)
- [ ] **공동인증서** 준비 (전자출원용)
- [ ] 지정상품 목록 최종 확정
- [ ] 상표 견본 준비 (한글 "낼름" + 영문 "Naelum")

### 4.2 출원 (1일)

- [ ] [특허로](https://www.patent.go.kr) 접속 → 상표출원서 작성
- [ ] 상표 유형: 문자상표 (한글+영문 병기)
- [ ] 상품류 및 지정상품 입력
- [ ] 출원료 납부 (전자결제)
- [ ] 출원번호 수령 → 보관

### 4.3 심사 대기 (약 12~18개월)

```
출원 → 방식심사 (1~2주) → 실체심사 대기 (12~15개월) → 실체심사 (1~2개월)
```

| 단계 | 소요기간 | 설명 |
|------|---------|------|
| 방식심사 | 1~2주 | 서류 형식 요건 확인 |
| 실체심사 대기 | 12~15개월 | 심사관 배정 대기 |
| 실체심사 | 1~2개월 | 식별력, 유사 여부 심사 |
| 출원공고 | 2개월 | 이의신청 기간 |
| 등록결정 | — | 등록료 납부 후 등록 |

### 4.4 우선심사 (선택)

심사기간을 **2~3개월로 단축** 가능

**조건:**
- 출원 상표를 이미 사용 중이어야 함
- 사용 증빙 자료 제출 (웹사이트 스크린샷, 서비스 화면 등)
- 우선심사 신청료: **160,000원** (전자출원 기준, 1류당)

**적용 가능성:** 서비스 런칭 후 신청 가능 (naelum.app 배포 후)

### 4.5 거절 시 대응

| 상황 | 대응 방법 |
|------|----------|
| 식별력 부족 거절 | 의견서 제출 — 사용에 의한 식별력 입증 |
| 유사상표 충돌 거절 | 의견서 제출 — 상품류 차이, 혼동 가능성 낮음 주장 |
| 지정상품 거절 | 보정서 제출 — 상품 범위 축소 |

## 6. 주의사항

1. **출원일 = 권리 발생 기준일**: 빨리 출원할수록 유리 (선출원주의)
2. **상표 사용 의무**: 등록 후 3년 이내 미사용 시 취소 가능
3. **등록료 납부 기한**: 등록결정 등본 수령 후 **2개월 이내** 미납 시 출원 포기
4. **43류 출원 자제**: 기존 "낼름" 등록 상표와 충돌 가능성 높음
5. **영문 표기 통일**: "Naelum"으로 일관되게 사용 (NELM, NELL REUM 등은 타인 상표)
