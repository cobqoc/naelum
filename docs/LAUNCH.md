## 🎯 포지셔닝 (중요)

**"만개의레시피 2025 버전"이 아니다.** 정면 경쟁은 100% 패배.

### Wedge (쐐기) 전략

**Primary 타겟**: 재한 외국인 + 해외교포 (8개 언어 지원 차별점)
- 재한 외국인 60만+, 해외교포 700만+
- 전용 앱 없음 = 빈 시장
- 8개 언어 이미 구현됨, 추가 비용 0

**Secondary 타겟**: MZ 1인 가구 요리 초보자
- 만개는 가족 중심, UI 복잡
- 낼름은 냉장고 기반 추천 + 쿠킹 모드 + 모바일 first

**메시지 예시**:
- EN: "Korean recipes you can actually follow — in your language"
- KO: "냉장고 열고 바로 만드는 한식"

### 만개의레시피 대비 우위 (경쟁 없는 영역)

| 낼름 무기 | 만개 상태 |
|---|---|
| 8개 언어 지원 | 한국어만 |
| AI 추천 (Gemini) | 전통 카테고리 |
| 모바일-first + 다크모드 | 데스크톱 시대 UI |
| 쿠킹 모드 + 음성 | 텍스트 나열 |
| 빠른 속도 (LCP 2.1s) | 느림 + 광고 |

### Anti-goal

- ❌ "만개보다 레시피 많이 만들기" — 절대 못 이김
- ❌ "한국어 유저 타겟" — 만개 영역
- ❌ "광고 수익" — 초기 아님
- ❌ "기능 완벽하게 만들기 후 런칭" — Perfectionism trap

## 🎬 유튜브 + AI 영상 전략

### 첫 영상 (T-7)
- **포맷**: 앱 소개 2분 영상
- **내용**: "낼름이 뭐하는 앱인가", "왜 만들었는가", "만개랑 뭐가 다른가"
- **CTA**: 영상 설명란에 naelum.app 링크 + 첫 레시피 링크

### 주기적 업로드 (주 3회 목표)
- **포맷**: 레시피 1개 = 영상 1개 (60초 쇼츠)
- **AI 워크플로우**:
  1. 레시피 DB에서 추천 레시피 선택 (조회수, 난이도 기준)
  2. Gemini로 영상 스크립트 생성 (기존 레시피 텍스트 입력 → 60초 대본)
  3. TTS로 음성 생성 (Google Cloud TTS or ElevenLabs)
  4. 레시피 이미지 + 음성 = 슬라이드쇼 영상
  5. 자동 업로드
- **설명란 템플릿**:
  ```
  🍳 자세한 레시피 & 재료로 추천 받기 → naelum.app/recipes/{id}
  📱 한국 음식을 여러 언어로 — 낼름 앱
  #한식 #요리 #레시피
  ```

### 영상 → 앱 유입 극대화
- [ ] 레시피 상세 페이지에 "유튜브 영상 보기" 버튼 추가 (레시피별 video_url 컬럼 활용)
- [ ] 홈 페이지에 "이번 주 영상" 섹션
- [ ] 영상 → 앱 딥링크 (모바일에서 YouTube → 브라우저 → 앱 설치 유도)

## 🧪 첫 유저 UX 감사 체크리스트

Day 1에서 확인할 항목:

### 첫 인상 (0-5초)
- [ ] 홈 페이지가 "뭐 하는 앱인지" 즉시 이해되나?
- [ ] "지금 가입해야 할 이유"가 명확한가?
- [ ] 로딩 속도가 Good 체감인가?

### 비로그인 유저 여정
- [ ] 비로그인으로 레시피 검색 가능한가?
- [ ] 레시피 상세 페이지 접근 가능한가?
- [ ] "저장하려면 로그인"이 자연스럽게 유도되나?

### 회원가입 플로우
- [ ] 회원가입 단계가 몇 개? (3개 초과면 이탈 급증)
- [ ] 이메일 인증 플로우 작동하나?
- [ ] OAuth (Google/Kakao) 버튼 작동하나?
- [ ] 온보딩(약관) 페이지가 산뜻한가?
- [ ] 첫 로그인 후 **어디로 보내는가**?

### 핵심 기능 경험
- [ ] 냉장고에 재료 추가 → 추천 받기 플로우
- [ ] 레시피 저장 → 낼름함 확인 플로우
- [ ] 요리 시작 → 쿠킹 모드 → 완료 플로우
- [ ] 검색 → 결과 → 상세 플로우

### 빈 상태 (Empty States)
- [ ] 냉장고 비었을 때 메시지가 친절한가?
- [ ] 저장된 레시피 없을 때 유도 문구?
- [ ] 검색 결과 없을 때 대안 제시?
- [ ] 리뷰 없을 때 "첫 리뷰 작성"?

### 에러 처리
- [ ] 잘못된 URL → 404 페이지 친절한가?
- [ ] 서버 에러 → error.tsx 친절한가?
- [ ] 네트워크 끊김 → 오프라인 페이지?
- [ ] 폼 검증 에러 메시지 명확한가?

### 다국어 (가장 중요)
- [ ] 영어로 홈 페이지 접속해봤을 때 자연스러운가?
- [ ] 다국어 전환 UI 눈에 띄나?
- [ ] 번역 품질 (기계 번역 티 나는지)?

### 공유 / 유입
- [ ] 레시피 URL 복사 → 카톡에 붙여넣으면 미리보기 뜨나?
- [ ] OG 이미지 정상 표시되나?
- [ ] Kakao 공유 버튼 작동하나?
